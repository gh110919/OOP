### Наследование

###### Позволяет создавать новые классы на основе существующих, наследуя их свойства и методы.

```
Представьте себе семью художников, 
где таланты и навыки передаются из поколения в поколение. 
Дедушка был талантливым живописцем, 
отец унаследовал его мастерство и добавил свои умения в скульптуре, 
а теперь внучка перенимает все эти навыки 
и также развивает их в области графического дизайна. 
Каждый последующий член семьи сохраняет основные черты своих предков, 
добавляя свои уникальные особенности. 
Так и в программировании: новый класс наследует свойства 
и методы существующего класса, 
но может также добавлять и свои уникальные черты.
```

```ts
class Employee extends Person {
  position: string;
  constructor(name: string, position: string) {
    super(name);
    this.position = position;
  }
  work() {
    console.log(`${this.name} работает как ${this.position}`);
  }
}

const employee = new Employee("Мария", "инженер");
employee.greet(); // Привет, меня зовут Мария
employee.work(); // Мария работает как инженер
```
